PrimerPy Manual

Using PrimerPy is as easy as:

  1. Paste your template sequence into the input box.
  2. Click the "Design" button and here you go!

More sophisticated?

Click the "More Options" button, you can specify a few options in the extended area.

There is now a module to check primer dimers:

Open menu PrimerPy/Check dimers and run it with any primers. It supports batch input and degenerate primers.
